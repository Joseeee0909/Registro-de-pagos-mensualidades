# 💰 Registro de Pagos y Mensualidades

Sistema web para el control de pagos mensuales, ingresos y seguimiento financiero de usuarios o grupos.

---

## 📌 Descripción

Esta aplicación permite registrar, organizar y visualizar pagos de mensualidades, facilitando el control de ingresos y el seguimiento de estados de pago. Está enfocada en simplificar la gestión financiera en pequeños sistemas o comunidades.

---

## ⚙️ Funcionalidades

* Registro de pagos mensuales
* Control de ingresos y estados de pago
* Visualización de datos financieros
* Gestión básica de usuarios o registros
* Organización de información por periodos

## 🚀 Instalación y ejecución

### 1. Clonar el repositorio

```bash id="mensa1"
git clone https://github.com/Joseeee0909/Registro-de-pagos-mensualidades.git
```

### 2. Instalar dependencias

```bash id="mensa2"
npm install
```

### 3. Ejecutar el proyecto

```bash id="mensa3"
npm run dev
```
